Java Calculator Application

Description

This is a simple Java Calculator Application developed using Java. It performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division through a menu-driven interface.

Features

- Addition of two numbers
- Subtraction of two numbers
- Multiplication of two numbers
- Division of two numbers
- Handles division by zero errors
- User-friendly menu system
- Option to perform multiple calculations

Technologies Used

- Java
- Scanner Class

How to Run

1. Open the project in any Java IDE (Eclipse, IntelliJ IDEA, VS Code, etc.).
2. Compile the "Calculator.java" file.
3. Run the program.
4. Select an operation from the menu.
5. Enter two numbers.
6. View the result.
7. Choose whether to continue or exit.

Sample Operations

- Addition
- Subtraction
- Multiplication
- Division
