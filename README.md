# Number Guessing Game - Java

## Description
This is a simple console-based Number Guessing Game developed in Java. The program generates a random number between 1 and 100, and the player has to guess the correct number within a limited number of attempts.

## Features
- Random number generation
- User-friendly console interface
- Feedback for each guess
  - Too High
  - Too Low
- Limited attempts
- Displays the correct number if the player loses
- Simple and easy-to-understand code

## Technologies Used
- Java
- Scanner Class
- Random Class

## How It Works
1. The program generates a random number between 1 and 100.
2. The user enters a guess.
3. The program checks the guess and provides feedback:
   - "Too High" if the guess is greater than the secret number.
   - "Too Low" if the guess is less than the secret number.
4. The game continues until:
   - The user guesses correctly, or
   - The maximum number of attempts is reached.
5. The program displays the result.
